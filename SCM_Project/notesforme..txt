Optional indicates that a variable can be either a specified type OR None.
Literal restricts a variable to only specific, exact values.
BaseModel is the foundation of Pydantic - it creates data validation classes. Think of it as a "supercharged" Python dataclass.
Field adds extra validation, metadata, and configuration to model fields.
